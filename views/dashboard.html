User Manual

These are few Guidelines for user:

1. Voter Registration

  • For casting the vote, user needs to first register himself. For this registration purpose, the user will be provided a voter registration form on this website.
  • The voter can only register in the registration phase. After the registration phase is over, the user cannot register and thus will not be able to vote.
  • For registration, the user will have to enter his identity card number and the account address which the user will be using for voting purpose./li>
  • At the first stage, the user's age will be checked for eligibility (18 years or older).
  • The second stage is OTP verification.

2. Voting Process

  • The voting process is divided into three phases.
  • 1. Registration Phase: During this phase, the registration of the users (which are going to cast the vote) will be carried out.
  • 2. Voting Phase: After initialization of voting phase from the admin, user can cast the vote in the voting section. The casting of vote can be simply done by clicking on “VOTE” button, after which transaction will be initiated, and after confirming transaction the vote will get successfully cast. After voting phase gets over user will not be able to cast a vote.
  • 3. Result Phase: This is the final stage of the whole voting process during which the results of election will be displayed in the “Result” section.
